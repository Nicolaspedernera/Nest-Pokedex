# Run in Dev mode.

1. Clone the repository.
2. Run:
```
npm install
```
3. Have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Run Data Base. 
```
docker-compose up -d
```

## Stack used.
* MongoDB
* Nestjs
